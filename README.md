JSON Typescript tools generated by ANTLR.

## Install / Import

```bash
yarn add json-antlr-tools-ts
```

```typescript
import * as JSONTools from "json-antlr-tools-ts";
// or
import {
	id,
	Lexer,
	Parser,
	Visitor,
	Listener,
	grammar,
	initialRule,
} from "json-antlr-tools-ts";
```
